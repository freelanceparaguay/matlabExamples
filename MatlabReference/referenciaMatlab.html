4 Funciones y procedimientos

Sentencia

 

Ejemplo

function ... end

Función en archivo .m

function [ salida1 salida2 ] = untitled2( entrada1, entrada2 )

  fprintf('entrada1= %.2f \n entrada2=%.2f \',entrada1, entrada2);

  salida1=10;

  salida2=20

end

 

Llamar desde la lines de comandos como

[a,b]=ejemplofuncion(1,2);

 

Los parámetros se devuelven a las variables encerradas entre corchetes.

@

Función en línea dentro de un script

%definición de función en línea

InterpolacionLinealGrado1 = @ (x0,fx0,x1,fx1,x) (fx0+((fx1-fx0)/(x1-x0))*(x-x0));

 

%llamado a función en línea

valorGrado1=InterpolacionLinealGrado1(1,3,5,99,4);
